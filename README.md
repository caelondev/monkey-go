# monkey-go

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

An implementation of Thorsten Ball's "Writing an Interpreter in Go" book with a slight modification. This project is a step-by-step walkthrough of the book's content, created as an educational exercise to learn about interpreter design and implementation.

## Overview

monkey-go is an interpreter for the Monkey programming language, a fictional language created by Thorsten Ball for his book. The goal of this project is to implement the interpreter in Go, following the book's guidance.

- Lexical analysis
- Parsing
- Evaluation
- Error handling

1. Clone the repository: `git clone https:
2. Navigate to the project directory: `cd monkey-go`
3. Run the interpreter: `go run main.go`

- Implementation based on Thorsten Ball's book: [Writing an Interpreter in Go](https://interpreterbook.com)
